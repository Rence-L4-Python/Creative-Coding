Primitive Painting Tool:

    CONTROLS:

  • Hold down both "S" key + right click to SMUDGE
  • Hold down "E" key + right click to ERASE
  • Hold down "Left CTRL" key to access EYEDROPPER + BRUSH MODE

    • EYEDROPPER + BRUSH MODE:

  • Hold down "Left CTRL" key + right click to select a color on the canvas
  • Hold down "Left CTRL" key + left click to draw the selected colors on the canvas

    TIPS:

  • Highly recommend using chromium-based browsers like Brave, Microsoft Edge, Opera, and Chrome to access a more modern colorpicker and eyedropper tool
  • When SMUDGING, you should use lower Brush Flow values
  • When using EYEDROPPER + BRUSH MODE, you can only use the colors that you selected while in the mode (not holding Left CTRL will use normal colors instead!)
